ActionBar — three states

Last updated April 25, 2026 · Spec for PR 2a

Scope. ActionBar mounts on project-scoped routes only — Hub, all subpages, and over open detail sheets. It does not mount on the projects root, which keeps its existing 5-arc fan and dashed "new project" tile.

Layout. Glass overlay style on every button. Back at bottom-left (36px), chat / + / threads cluster centered (chat 36 · + 52 · threads 36, gap 10), resources at bottom-right (36px). Bar bottom inset 18px.

+ slot. When + hides on a detail sheet, the slot stays empty — cluster width remains uniform across states.

Behavior. back is contextual (closes top-of-stack: detail sheet → page; subpage → Hub; Hub → projects list). chat, threads, resources always navigate to global project-scoped views regardless of surface. + behavior is registered per-page via FabActionContext.
