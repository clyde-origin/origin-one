# One Lore

World-building, character, mythology. The creative source.
STATUS: Stub preserved from early exploration. Not in active development.

## What this app does
- Character profiles and relationships
- Location database
- Mythology, tone, world rules
- Lite writing / scene capture (saves to shared Document schema)

## What this app does NOT do
- No shotlisting (that's One Arc and Back to One)
- No storyboarding (that's One Arc and Back to One)
- No production tools of any kind

## Key behavior
A scene written in One Lore saves to the shared Document schema.
It appears in One Arc and Back to One automatically — no export needed.

## When to build
After One Arc ships and is proven.
